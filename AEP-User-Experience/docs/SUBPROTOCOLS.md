# AEP Subprotocols (2.8)

Seven regulation subprotocol crates live under [`AEP-Subprotocols/`](../../AEP-Subprotocols/). Unified CLI: `aep-subprotocol`.

| Crate | Domain |
|-------|--------|
| `aep-subprotocol-ui` | UI element governance |
| `aep-subprotocol-commerce` | Commerce flows |
| `aep-subprotocol-workflows` | Workflow orchestration |
| `aep-subprotocol-rest-api` | REST API contracts |
| `aep-subprotocol-events` | Event streams |
| `aep-subprotocol-iac` | Infrastructure as code |
| `aep-subprotocol-coding-governance` | Coding governance announcements |

See [`AEP-Subprotocols/README.md`](../../AEP-Subprotocols/README.md) for catalog and build instructions.
