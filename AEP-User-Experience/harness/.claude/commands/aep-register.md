# /aep-register

## AEP 2.8 Element Registration

Same three-file registration as 2.75e: `aep-registry.yaml`, `aep-scene.json`, `aep-theme.yaml`.

For Composer Lite canvas nodes, also check `AEP-Base-Node/registry/catalog.json` if adding a registry-backed node type.

After registration run `/aep-validate`.
